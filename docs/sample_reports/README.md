# ReleaseGuard Sample Reports

These are curated static report examples generated from ReleaseGuard fixtures.
They let reviewers inspect the product output without running the CLI.

Each sample contains:

- `report.md` for GitHub-friendly reading.
- `report.html` for the static HTML artifact generated by ReleaseGuard.

The reports are sanitized examples. Runtime artifact paths use
`artifacts/releaseguard/RUN_ID`, and test durations use `<duration>ms`.

## Regenerate

From the repository root:

```bash
./scripts/generate_sample_reports.sh
```

Then run the normal verification suite:

```bash
./scripts/verify_releaseguard.sh
```

Do not commit raw `artifacts/releaseguard` run directories. Only commit curated
files under `docs/sample_reports`.
